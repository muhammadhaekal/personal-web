## Simple Personal Website

Live Demo: http://personal-web-hkl.netlify.com

## Tech Stacks

- ReactJs
- Redux
- Styled-components

## How to run this project

### 1. Install All Setup Requirement

- NodeJs
- NPM

### 2. Run This Command on Project Folder

```
# Development
npm i
npm start

# Production
npm i
npm build
```
